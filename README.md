# Advance_Calculator
Advance Calculator using C++. 
<br>
"Overview"
The Advanced Scientific Calculator is a console-based C++ application designed to perform a wide range of mathematical operations. It combines basic arithmetic, scientific calculations, matrix operations, statistics, and calculus into a single unified system.

This project demonstrates strong concepts of Object-Oriented Programming (OOP), data structures, and mathematical computation.
<br>
"Objectives"
To develop a multi-functional calculator using C++
To implement real-world mathematical computations
To strengthen understanding of OOP concepts
To provide a user-friendly menu-driven interface
<br>
"Technologies Used"
Programming Language: C++
Concepts Used:
    Object-Oriented Programming (Classes & Objects)
    STL (Vectors, Stack)
    File Handling
    Mathematical Libraries (<cmath>)
<br>
"Features"
1. Basic Operations
    Addition, Subtraction, Multiplication, Division
    Square, Cube, Square Root
    Factorial Calculation
2. Trigonometric Functions
    sin, cos, tan, cot, sec, cosec (in degrees)
3. Logarithmic Functions
    Log base 10
    Natural Log (ln)
4. Statistics
    Mean
    Variance
5. Matrix Operations
    Addition
    Subtraction
    Multiplication
    Division (using inverse)
    Transpose
    Inverse (2x2)
    Determinant (NxN)
6. Number System Conversions
    Binary ↔ Decimal
    Decimal ↔ Hexadecimal
    Binary ↔ Hexadecimal
    Octal conversions
7. Memory Functions
    Memory Add
    Memory Recall
    Memory Clear
8. Expression Parser
    Evaluates mathematical expressions using stacks
    Supports parentheses and operator precedence
9. Equation Solver
    Linear equations (ax + b = 0)
    Quadratic equations (ax² + bx + c = 0)
10. Calculus
    Numerical Differentiation
    Numerical Integration (Trapezoidal Rule)
11. Prime Number Checker
    Checks whether a number is prime
12. History Feature
    Stores calculation history in a file (history.txt)
    Allows users to view past calculations
<br>
"Working Principle"

The calculator is implemented using a menu-driven approach. The main menu allows users to navigate through different functionalities. Each feature is implemented as a member function of the Calculator class.

    Input is taken from the user via console
    Processing is done using mathematical logic and algorithms
    Output is displayed instantly
<br>
"Key Concepts Used"
Encapsulation: All functionalities are wrapped inside a class
Modularity: Each operation is defined as a separate function
